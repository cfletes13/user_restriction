OVERVIEW

This application is useful to add restriction of creation new product and partner from specific user


Work With Both Community And Enterprise Version.

KEY FEATURES:

  • Easy To Use.
  • Restrict New Partner Creation
  • Restrict New Product Creation
  • User Wise Setting

CONFIGURATION

Goto Settings --> Users --> Users --> Access Rights Page --> Technical Settings --> Restrict To Create Product And Partner

DETAILS

Create button is invisible for Customer/Vendor kanban, list, form view. When user select partner from SO, PO, INV still Partner creation option is invisible.

Create button is invisible for Product/Variants kanban, list, form view. When user select product from SO, PO, INV still product creation option is invisible.
